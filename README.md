# Shopee Affiliate API Proxy

Backend para o Gerador de Links Shopee Afiliado.

## Deploy no Vercel

1. Acesse vercel.com e faça login com GitHub
2. Clique em "Add New Project"
3. Importe este repositório
4. Clique em "Deploy"
5. Copie a URL gerada (ex: shopee-api.vercel.app)
6. Cole na configuração da ferramenta

## Endpoints

POST /api/shopee — Proxy para a API Shopee Afiliados
